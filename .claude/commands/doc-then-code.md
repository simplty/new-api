# 文档优先开发流程

**功能需求**：$ARGUMENTS

---

## 📝 阶段一：文档更新

我现在将为功能「$ARGUMENTS」更新相关文档。

### 需要更新的文档：

1. **自定义功能文档.md**
   - 添加新功能的概述
   - 更新使用说明
   - 添加示例代码

我将立即开始更新这些文档。更新完成后，请查看文档内容。

---

## ⏸️ 确认点

文档更新完成后，请检查：
- 功能描述是否准确
- 使用说明是否清晰
- 示例是否完整

**如果文档满意，请回复："文档确认，开始实现代码"**
**如果需要修改，请直接告诉我需要调整的内容**

---

## 💻 阶段二：代码实现

收到你的确认后，我将基于文档实现代码：

1. **核心功能实现**
   - 根据文档描述实现功能
   - 保持代码与文档一致
   - 添加必要的注释

2. **测试编写**
   - 单元测试
   - 集成测试（如需要）
   - 测试文档中的示例

3. **代码质量检查**
   - 运行 lint 检查
   - 运行类型检查
   - 确保测试通过

---

## 🎯 使用方法

在 Claude Code 中使用：
```
/doc-then-code 添加用户认证功能
```

然后按照流程指引操作即可。
